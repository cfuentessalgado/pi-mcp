# Security Policy

**Please do not disclose security-related issues publicly.** Use the private reporting channel below so maintainers can triage and release a fix before details are public.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately using **GitHub Private Vulnerability Reporting**:

1. Open the repository's **Security** tab.
2. Click **Report a vulnerability**.
3. Include affected versions, impact, reproduction steps, and any proof-of-concept details.

Maintainers will acknowledge reports as soon as practical, investigate, and coordinate remediation through the private advisory workflow.

## Supported Versions

This project is currently pre-1.0. Security fixes are provided for the latest release only.
